# Dog AI

Reconhecimento de raças de cães utilizando Tensorflow & TeachableMachine

Demo: [https://dogai.netlify.com](https://dogai.netlify.com)

## Quick Start

```
$ git clone https://github.com/jeferson-sb/dogai.git && cd dogai
```

```
$ yarn
```

```
$ yarn start
```
